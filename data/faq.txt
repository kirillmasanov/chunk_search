Вопрос: Можно ли работать с Yandex Cloud AI Studio без интернета?
Ответ: Нет, для работы с Yandex Cloud AI Studio требуется постоянное подключение к интернету, так как все вычисления происходят в облаке. Модели и данные хранятся на серверах Yandex Cloud, и для выполнения запросов необходим доступ к API.
Вопрос: Какие модели доступны в AI Studio?
Ответ: В Yandex Cloud AI Studio доступны следующие модели: YandexGPT для сложных задач, YandexGPT-Lite для быстрых ответов, qwen3-235b для продвинутой генерации текста, и специализированные модели для embeddings.
Вопрос: Как создать Vector Store?
Ответ: Для создания Vector Store используйте метод create() через OpenAI SDK. Укажите имя хранилища и список файлов для индексации. После создания дождитесь статуса completed, проверяя его через метод retrieve().
Вопрос: Что такое чанкование документов?
Ответ: Чанкование - это процесс разбиения больших документов на меньшие фрагменты (чанки) для эффективного поиска и обработки. Можно использовать автоматическое чанкование платформы или загрузить предварительно подготовленные чанки в формате JSONL.
Вопрос: Какой формат данных поддерживается для пользовательских чанков?
Ответ: Для загрузки пользовательских чанков используется формат JSONL (JSON Lines). Каждая строка файла должна содержать JSON объект с обязательным полем body, содержащим текстовое содержимое чанка. При загрузке необходимо указать extra_body с параметром format равным chunks.
